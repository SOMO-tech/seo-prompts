# Robots.txt with AI Bot Controls

Create a robots.txt with explicit rules for AI and LLM crawler bots, in addition to standard search engine rules.

## Standard search engines

Googlebot, Bingbot, and other major crawlers get full access to public content.

## Disallow for all bots

- API routes
- Admin areas
- User private data paths
- Duplicate content paths

## Sitemap

Include the sitemap reference.

## Documentation

Add clearly commented sections explaining each rule block so non-technical team members can understand the file.
